# NeuralSynk SpA — Sitio web

Landing page de **NeuralSynk SpA** — servicios de backend, frontend, datos con IA y nube.

Sitio estático (HTML, CSS y JS, sin dependencias). Desplegado en Cloudflare Pages.

## Estructura
- `index.html` — página principal
- `styles.css` — estilos y efectos
- `script.js` — fondo de red neuronal e interacciones

## Desarrollo local
Abre `index.html` en el navegador, o levanta un servidor:

```bash
python3 -m http.server 8000
# luego abre http://localhost:8000
```
